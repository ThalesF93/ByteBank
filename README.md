ByteBank 💰

Sistema bancário desenvolvido para simular operações financeiras básicas, com foco em aprendizado de programação orientada a objetos, organização de código e boas práticas de desenvolvimento.

📖 Sobre o Projeto

O ByteBank é uma aplicação criada com o objetivo de representar funcionalidades comuns de um banco digital, permitindo operações como:

Criação de contas
Depósitos
Saques
Transferências
Gerenciamento de clientes
Controle de saldo

O projeto foi desenvolvido para fins educacionais e prática de conceitos de desenvolvimento backend.
